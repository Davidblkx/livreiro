# Livreiro

Codigo fonte para o site [livreiro.pt](https://livreiro.pt)

## Instalação

### Requisitos

- [Deno](https://deno.land/)
- [git](https://git-scm.com/)

### Correr o site

- `git clone https://github.com/Davidblkx/livreiro.git`
- `cd livreiro`
- `deno run -A ./dev.ts`
- Abrir o website em `http://localhost:8666`
